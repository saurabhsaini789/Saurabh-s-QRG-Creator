# Saurabh's QRG Creator

A professional tool for creating Quick Reference Guides (QRG) with ease.

## Features
- **Branding**: Upload your company logo and customize colors.
- **Rich Text**: Full-featured text editor for step descriptions.
- **Drag & Drop**: Easily reorder steps.
- **Export**: Generate high-quality PDFs (Letter, A4, Legal sizes).
- **Live Preview**: See changes in real-time.

## Tech Stack
- React 19
- Vite
- Lucide React
- Dnd Kit
- html2pdf.js

## Deployment
Deployed via GitHub Actions to GitHub Pages.
